/* 对于响应式布局，现代化的处理方式是使用flex布局或是grid布局，这两种布局都是基于现代浏览器的CSS3标准来实现的 */ /* 相对更早一点的方法是使用@media来实现响应式布局，这种方法的缺点是不能实现灵活的布局，因为它只能在一个屏幕的宽度范围内进行布局 */ /* 对图片实现响应式的方法， 可以对不同分辨率提供不同的尺寸图片，即指向多个源文件 */ /* srcset属性定义了允许浏览器选择的图像集，以及每个图像的大小 */ /* sizes定义了一组媒体条件， 例如当屏幕的宽度小于600px时， 图片的宽度就是480px，反之则为800px */ /* 这种响应式的目的首先是为了让移动端客户节省宽带，防止图片过大，导致浏览器做一些额外的工作，从而导致页面的加载速度变慢 */ /* 相同大小，不同分辨率 */ /* picture是一个包装器，为浏览器提供不同的来源以供选择 */ /* media包含媒体条件属性，决定显示哪个图片 */ /* img是一个必须要提供的元素，否则不会出现任何图片，这表示没有任何媒体条件返回true时，显示该图片 */
